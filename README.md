# Vue - DRF

## 1. 기본 설정

1. Django

   1. 가상환경 설정

      ```bash
      $ python -V
      $ python -m venv venv
      $ source venv/Scrtipts/activate
      ```

      

   2. 패키지 설치

      ```bash
      $ pip install django djangorestframework
      $ pip freeze > requirement.stx
      ```

   3. django 프로젝트 설정

      ```bash
      $ django-admin startproject {프로젝트명}.
      ```

      

2. Vue

   1. VueCLI 설치

      ```bash
      $ npm insatll -g @vue/cli
      ```

   2. Vue 프로젝트 설정

      ```bash
      $ vue create {프로젝트 이름}
      ```
